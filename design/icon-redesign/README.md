# Flow App 图标重绘交接包

这个文件夹集中保存当前源码引用的图标，供重新设计后原名覆盖。

## 使用约定

1. 请直接编辑 `icons/` 中的 SVG，并保持文件名不变。
2. 建议保留原有 `viewBox`，图形尽量使用 `currentColor`，不要写死界面颜色。
3. 线性图标建议统一端点、转角和视觉粗细；品牌对象图标可以保持填充风格。
4. `usage.csv` 记录图标来源与当前用途。标为 `ui-primitive` 的图标属于通用控件，标为 `product` 的图标是产品界面直接使用的图标。
5. 你完成重绘后，我会按同名文件接入源码，并检查尺寸、对齐、选中态和无障碍标签。

## 建议的统一规则

- 基准画布：24 × 24
- 常规描边：2px
- 圆角端点与圆角连接
- 产品图标不要混用轮廓、双色、拟物和不同透视角度
- `FlowSource.svg`、`Flow.svg`、`Fragment.svg` 是“源 / 流 / 灵感碎片”三个核心对象，允许使用统一的品牌填充语言

当前界面仍使用原实现；这个交接包暂时不会自动影响 App。
